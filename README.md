# repack - 無圧縮 ZIP 変換ツール

圧縮ファイルを **無圧縮 ZIP (Store)** に変換する SendTo ツールです。

## 機能

- ZIP / RAR / 7z / tar.gz / CBZ / CBR など主要形式に対応
- ZIP 内のゴミファイルを設定ファイルで定義して自動削除
- **未分類ファイルはユーザーに確認し、判断を自動保存（decisions.json）**
- **文字コード問題に対応（Shift-JIS / UTF-8 / その他のエンコーディング）**
- ルートに単一フォルダしかない場合、フォルダを剥がしてフラット化
- 元ファイルの更新日時を引き継ぐ
- 元ファイルはゴミ箱へ送る（完全削除ではない）
- 複数ファイルの一括処理対応
- エラーがなければ 3 秒後に自動終了

## セットアップ手順

### 1. Python と 7-Zip をインストール

- Python: インストール時に **「Add Python to PATH」にチェック**を入れること
- 7-Zip: 標準の場所 (`C:\Program Files\7-Zip\`) にインストール

### 2. ビルド

```
build.bat をダブルクリック
```

`dist\repack.exe` と `dist\config.toml` が生成されます。

### 3. SendTo に登録

```
install_sendto.bat をダブルクリック
```

### 4. 使用方法

エクスプローラーで圧縮ファイルを選択（複数可）→ 右クリック → 「送る」
→ **「repack (無圧縮ZIP変換)」**

## ファイル構成

```
repack.py              ← メインスクリプト
config.toml            ← 静的設定ファイル（手動編集）
build.bat              ← exe ビルド用バッチ
install_sendto.bat     ← SendTo 登録バッチ

dist/
  repack.exe           ← 生成される実行ファイル
  config.toml          ← コピーされた設定ファイル（こちらを編集する）
  decisions.json       ← 対話的判断の永続ストア（自動生成・自動更新）
```

## ゴミファイルの管理

### 自動学習（decisions.json）

初回実行時、ルールに一致しない「未分類ファイル」が見つかると確認プロンプトが表示されます。

```
  ┌─ 未分類ファイル: info.txt
  │  このファイルは既存のルールに一致しませんでした。
  │  選択した内容は decisions.json に保存され、次回から自動適用されます。
  ├─ [K] 保持する（allow_patterns に追加）
  └─ [D] 削除する（junk_patterns に追加）
  選択 [K/D]:
```

一度決定した内容は **`decisions.json`** に保存され、次回以降は確認なしに自動適用されます。

### 手動設定（config.toml）

`dist\config.toml` を任意のテキストエディタで編集します。

```toml
# 未分類ファイルの処理方針: "ask" / "keep" / "junk"
unknown_file_action = "ask"

# 保持を強制するパターン（junk_patterns より優先）
allow_patterns = [
    # "cover.jpg",
]

# ゴミとして削除するファイル名パターン (glob形式)
junk_patterns = [
    "*.url",
    "Thumbs.db",
    # ...
]

# ゴミとして削除するディレクトリ名パターン
junk_dirs = [
    "__MACOSX",
    # ...
]
```

### decisions.json のリセット

判断を最初からやり直したい場合は `decisions.json` を削除してください。

## 処理の流れ

```
SendTo で起動
  ↓
7z.exe で一時フォルダに展開
  ↓
config.toml のパターンに合うファイル・フォルダを削除
  ↓
ルートに単一フォルダのみ → そのフォルダを新ルートにする（繰り返し）
  ↓
zipfile モジュールで無圧縮 ZIP を生成
  ↓
元ファイルをゴミ箱へ
  ↓
同じ場所に .zip で保存
  ↓
元ファイルのタイムスタンプを適用
```

## トラブルシューティング

**「7-Zip が見つかりません」と表示される**
→ 7-Zip を標準の場所にインストールするか、`repack.exe` と同じフォルダに `7z.exe` を置く。

**RAR ファイルが展開できない**
→ 7-Zip の最新版を使用してください。RAR5 形式は 7-Zip 最新版が必要です。

**日本語ファイル名が文字化けする**
→ v2.0 からは 7-Zip に `-mcp=932` を渡して Shift-JIS ファイル名を自動処理します。
  改善されない場合は 7-Zip を最新版に更新してください。

**コンソールの文字が化ける**
→ v2.0 からは起動時に UTF-8 コンソールに切り替えています。
  それでも問題がある場合は `chcp 65001` を実行してから repack.exe を起動してください。

**一度「削除」にした判断を取り消したい**
→ `decisions.json` をテキストエディタで開いて該当エントリを削除するか、
  ファイル全体を削除して次回起動時に再判定させてください。
